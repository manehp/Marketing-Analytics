# Telco Customer Churn & CLV Analysis

This project analyzes customer churn and calculates Customer Lifetime Value (CLV) for a telecom company using survival analysis models. The goal is to understand factors affecting churn risk, identify valuable customer segments, and estimate retention budgets.

## Repository Contents
- `code_report.ipynb` : Jupyter notebook containing all code, visualizations, explanations and report
- `requirements.txt` : Python dependencies for running the notebook
- `README.md` : Project overview
